# ScoreLink 수동 시나리오 체크리스트 (P10)

`npm run build && npx vite preview`로 dist를 서브한 뒤 실기기/브라우저에서 점검한다.
자동화된 검증(테스트 250개, 골든 스냅샷 16종)이 커버하지 못하는 실감각 항목 중심.

## 1. 편집 — 입력과 소리

- [ ] 첫 탭에서 오디오가 활성화된다 (iOS/Android: 무음 모드 아님 확인)
- [ ] 패드 빈 칸 탭 → 4분음표 입력 + 프리뷰 사운드 + 악보 즉시 반영
- [ ] 켜진 칸 탭 → 선택(프리뷰), 재탭 → 삭제
- [ ] 쉼표 행: 음 머리 탭=삭제, 중간 탭=자르기, 빈 곳 탭="이미 쉼표입니다" 토스트
- [ ] 음높이 스테퍼로 ♯음 생성 → 패드에 ♯행 등장, 행에서 직접 입력 가능, 마지막 ♯음 삭제 시 행 소멸
- [ ] 위치/길이 스테퍼 변경 시 무음(프리뷰 없음), 겹침 자동 해소
- [ ] 마디 걸침 음: 패드 마지막 칸 ⤳, 악보 분할+타이
- [ ] undo/redo 60단계, 실행 후 선택/마디 정합

## 2. 편집 — 재생

- [ ] 전체 재생: 현재 마디 따라가며 줄 자동 전환, 플레이헤드 부드럽게 진행
- [ ] 마디 재생(▶마디): 현재 마디 범위만
- [ ] 재생 중 반주 패턴 변경/끔 → 현재 위치부터 실시간 반영, 팝 노이즈 없음
- [ ] 재생 중 메트로놈 토글 → 켜면 남은 박만, 끄면 클릭만 즉시 정지(멜로디 유지)
- [ ] 정지 시 즉시 무음, 아이콘 play로 복귀
- [ ] 반주 3종 청감: pad 지속, comp 박마다, arp 8분 순환 (`npm run dump`와 대조 가능)
- [ ] 메트로놈 액센트(정박 1568Hz)와 pickup 박 무액센트

## 3. 코드/템포/마디

- [ ] 코드 슬롯 탭 → 피커 즉시 적용, 지우기, 악보 위 코드 텍스트 표시
- [ ] 마디별 반주 버튼 순환: 기본→패드→컴핑→아르프→끔→기본
- [ ] 템포 직접 입력 40~240 클램프, ±4, 칩 동기화
- [ ] 마지막 마디 ▶=＋ 마디 추가 / 첫 마디 ◀=＋ 못갖춘 추가(2박), 중복 추가 불가
- [ ] 못갖춘마디: 패드 8열(박 3·4), 악보 첫 줄 70% 폭 마디

## 4. 공유/열람/영속성

- [ ] 편집 후 새로고침 → 곡 복원 (자동 저장 500ms)
- [ ] 공유 버튼 → 클립보드에 `?mode=view#v1.…` URL, 토스트
- [ ] 공유 링크를 새 탭/다른 기기에서 열기 → 열람 화면 + 동일 곡
- [ ] 구버전 프로토타입 해시(`fixtures/prototype-hashes.json`의 legacy)를 URL에 붙여 열기 → 정상 렌더
- [ ] 열람: 배지 4종, 멀티라인 스크롤, 음표 탭 → 그 위치부터 멜로디만 재생
- [ ] 복제해서 편집 → 편집 화면 전환 + 데이터 유지

## 5. 실측 항목 (기기 필요 — 미완)

- [ ] **iOS Safari URL 해시 길이 한계 실측**: 최장 곡(마디 최대 + 노트 최대)을 만들어 공유 URL 길이 확인, Safari 주소창/메시지 앱 전달 시 잘림 여부 (harness.md §9-4)
- [ ] Android Chrome에서 SMP 음악 기호 미사용 확인 (악보가 □로 깨지지 않음 — 설계상 보장, 시각 확인만)
- [ ] Lighthouse 모바일 성능/접근성 90+ (참고치)
- [ ] 실기기 오디오 레이턴시/팝 노이즈 청감 확인

## 데스크톱 DAW 편집 (P11 v2)

- [ ] 창 폭 <900px에서 모바일 레이아웃과 동일(회귀 없음)
- [ ] 창 폭 ≥900px 편집: 다크 DAW(트랜스포트 바·코드 레일·피아노롤·인스펙터·하단 악보·상태 바)
- [ ] 피아노롤: 빈 셀 클릭=입력, 노트 클릭=선택, 선택 재클릭=삭제 (모바일 패드와 동일 의미론)
- [ ] 피아노롤 재생 중 레드 플레이헤드 진행
- [ ] 인스펙터: 음높이/위치/길이 ± 편집, 반주 전역·마디 오버라이드
- [ ] 코드 레일 슬롯 클릭 → 피커 열림/적용, Esc 닫힘
- [ ] 하단 악보 마디 클릭 → 현재 마디 이동
- [ ] `?mode=view` 열람이 중앙 680px 라이트 컬럼
- [ ] 키보드: ←→/hl 선택, ↑↓/kj 음높이, Shift 위치·길이
- [ ] 한글 입력 모드에서도 hjkl(ㅗㅓㅏㅣ 자리)·Cmd+Z 동일 동작
- [ ] 키보드: Space 재생, Enter 마디재생, Delete 삭제, [ ] 마디이동, Cmd/Ctrl+Z 실행취소, Esc 피커닫기
- [ ] 입력창(템포) 포커스 중에는 단축키 미작동
- [ ] 창 크기를 900px 경계로 넘나들 때 편집 상태(곡·선택) 유지

## 참고
